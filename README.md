- 技术栈：vue 全家桶 + axios + dexie + typescript
- 功能列表：

* [x] 华丽首页展示
* [x] 博客列表
* [x] 博客详情
* [ ] 音乐
* [ ] 示例
* [ ] 关于我

## 2.server

- 技术栈：koa 全家桶 + puppeteer-core + nod + typescript

- 功能列表：

* [x] 根据 IP 定位当前城市
* [x] 当前天气及未来一周天气详情
* [x] 通过 puppeteer 爬取最新业界动态文章
* [ ] 爬取的信息存储到本地文件，并存储在 IndexedDB 中
